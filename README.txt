== Description ==

This plugin sends emails to remind subscribers that their subscription will be renewed in X days.

== Installation ==

- run "xgettext" in order to detect translations
- Upload `subscription-renewal-reminder.php` to the `/wp-content/plugins/` directory.
- Activate the plugin.
